## git 命令使用练习

    - 新建文件夹
        > mkdir 文件夹名称
    - 创建文件
        > touch index.js
    - 查看文件内容
        > cat index.js
    - 编辑文件
        > vi index.js

## git 分区管理
    - 添加到暂存区
        > git add .
    - 从暂存区添加到历史区
        > git commit -m'提交日志'
    - 查看文件提交状态
        > git status (红色是在工作区文件，绿色是在暂存区的文件)
